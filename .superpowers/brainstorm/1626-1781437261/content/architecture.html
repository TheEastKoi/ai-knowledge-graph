系统架构设计

Vue 3 + Vite + TypeScript + Pinia 技术栈，赛博学术风视觉

技术栈

Vue 3 (Composition API) + TypeScript + Vite Pinia (状态管理) Vue Router (路由) D3.js v7 (知识图谱) Prism.js (代码高亮) Tailwind CSS v4 (样式) Motion (微动效)

不引入重型 UI 库，以 Tailwind + 自定义组件实现赛博学术风，保持轻量

路由设计

路由视图说明
/GraphView默认：交互式知识图谱（保留 D3 力导向图）
/listListView卡片/列表浏览，支持分类过滤和搜索
/roadmapRoadmapView分阶段学习路线图，追踪学习进度
/interviewInterviewView面试题库，按主题 / 难度分类
/detail/:idDetailView知识点详情页：完整解释 + 代码 + 公式 + 关联

组件树

AppLayout
全局布局壳：导航栏 + 主内容区 + 底部状态栏。赛博学术风顶部导航
NavBar
靛蓝主题导航：Logo、路由标签、搜索入口、模式切换（引导/面试）
KnowledgeGraph
D3.js 力导向图核心组件。5层分层布局、拖拽、缩放、节点高亮
DetailPanel
侧边栏详情：定义、地位、前置、下一步、关联标签（图谱视图中）
DetailPage
独立详情页：完整 Markdown 内容、代码示例、公式、Mermaid 图表
ConceptCard
知识卡片组件：层级标签、分类色标、简述、展开按钮
ConceptList
列表视图：虚拟滚动、分层过滤、分类标签、搜索高亮
LearningRoadmap
5 层学习路线：进度条、已完成/进行中/未开始状态
InterviewPanel
面试模式：四阶段分类、题目卡片、答案展开、难度标签
CodeBlock
代码展示：Prism.js 语法高亮、复制按钮、语言标签
SearchBar
全局搜索：命令面板风格 (Ctrl+K)、模糊匹配、即时预览
LevelFilter
层级筛选器：5 层切换、分类筛选 (LLM/Agent/基础)

数据流

1
knowledgeStore (Pinia) — 全局知识数据仓库。启动时加载完整的知识 JSON，包含 33+ 节点的完整信息：定义、公式、代码示例、面试题
2
appStore (Pinia) — 应用状态：当前模式 (normal/guide/interview)、选中节点、学习进度 (localStorage 持久化)、搜索查询
3
组件消费 — GraphView 读取完整数据渲染图谱；ListView 读取过滤后数据；DetailPage 根据路由参数查找单条数据；RoadmapView 结合进度数据展示
4
响应式联动 — 点击图谱节点 → 更新选中状态 → DetailPanel 响应 → 同时更新 URL hash。路由切换 → 图谱定位到对应层级

知识数据扩展计划

每个概念节点从现有的 4 个字段扩展到完整的学习资料：

字段现有扩展后
核心定义简短描述500-800 字深度解释，含类比、图示
公式 / 原理无关键数学公式 (LaTeX)、算法伪代码
代码示例无Python/JS 实战代码（LangChain、transformers 等）
架构图无Mermaid 流程图 / 时序图
面试题仅面试模式标签每个概念 2-4 道常见面试题 + 参考答案
最佳实践无生产环境经验、常见陷阱
延伸阅读无论文链接、官方文档、推荐博客
难度等级无入门 / 中级 / 高级，用于过滤和学习路线
